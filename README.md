## 🇫🇷 WakfuGoultard

**WakfuGoultard** est un projet scolaire réalisé en groupe ayant pour objectif de créer un site web dédié à un personnage de notre choix. Notre choix s'est porté sur **Goultard**, personnage emblématique de l'univers **Wakfu**.

### 🎯 Objectifs du projet

Le projet imposait un cadre technique précis :

- ⚛️ Utilisation de **React** (front-end uniquement, pas de back-end)
- 🗺️ Navigation entre plusieurs pages avec **React Router**
- 📅 Intégration d'une **chronologie** imposée dans le cahier des charges
- 🎨 Utilisation de **Tailwind CSS** pour le style, avec une liberté totale sur l'esthétique

### 🎨 Notre approche

Dans le respect des contraintes imposées, nous avons eu carte blanche sur l'identité visuelle du site. Nous avons donc fait nos propres choix artistiques pour rendre hommage au personnage de Goultard à notre façon.

### 🌍 Langue

Le site est actuellement disponible **uniquement en français**.

### ⚠️ Version scolaire

> Ce projet a été réalisé dans un cadre scolaire. Il s'agit d'une version figée correspondant au rendu final du projet de groupe.

## 🇬🇧 WakfuGoultard

**WakfuGoultard** is a school group project with the goal of creating a website dedicated to a character of our choice. We chose **Goultard**, an iconic character from the **Wakfu** universe.

### 🎯 Project objectives

The project came with a strict technical framework:

- ⚛️ Built with **React** (front-end only, no back-end)
- 🗺️ Multi-page navigation using **React Router**
- 📅 Integration of a **timeline** as a mandatory feature
- 🎨 Styling with **Tailwind CSS**, with full creative freedom on aesthetics

### 🎨 Our approach

Within the given constraints, we had complete freedom over the visual identity of the site. We made our own artistic choices to pay tribute to Goultard in our own way.

### 🌍 Language

The site is currently available **in French only**.

### ⚠️ School project

> This project was completed as part of a school assignment. It is a frozen version corresponding to the final group submission.
